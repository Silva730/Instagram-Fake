# Ferramenta Educacional de Phishing — Simulação (Python)

Este projeto é uma **ferramenta educacional**, desenvolvida em **Python**, com o objetivo de **estudo e conscientização em segurança da informação**.  
Ele simula uma página inspirada no Instagram para **análise de vetores de ataque e defesa**, **exclusivamente em ambiente controlado**.

## Objetivo

Demonstrar como ataques de phishing funcionam do ponto de vista técnico, permitindo estudar:
- Detecção
- Mitigação
- Boas práticas de proteção
- Análise de logs e tráfego

## Escopo e uso responsável

- Projeto voltado **apenas para fins educacionais**
- Utilização **restrita a laboratório/testes autorizados**
- Não destinado a uso malicioso ou em ambientes reais
- Foco em aprendizado ofensivo **para fortalecimento da defesa**

## Características gerais

- Simulação de página de login para estudo
- Captura de eventos em ambiente controlado
- Integração com infraestrutura de proteção (ex: Cloudflare) para análise de comportamento
- Código escrito do zero para fins didáticos

## Tecnologias utilizadas

- Python
- HTML / CSS (interface simulada)
- Infraestrutura de proteção e proxy reverso (Cloudflare)

## Observações importantes

- Projeto **não deve ser usado contra terceiros**
- Todo o conteúdo é para **pesquisa, estudo e treinamento**
- O autor não incentiva nem apoia atividades ilegais

## Status do projeto

- Tipo: Ferramenta educacional
- Área: Segurança da Informação
- Estado: Funcional em ambiente de teste
